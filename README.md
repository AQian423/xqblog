## 简介

个人博客。

## 结构

```
qwz-blog
├─blog-backend  后端源码
│
├─blog-frontend 前端源码
│  ├─blog-frontend-admin 后台管理
│  ├─blog-frontend-portal 前台门面
│ 
├─blog-resource 项目资源
│  ├─docker 容器部署
│  ├─image 图片
│  ├─jar 依赖包
│  ├─logs 项目日志
│  ├─sql 数据库文件
│  ├─static 静态资源
```
