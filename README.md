## 1、简介

根据个人需求，基于 Halo-1.4.2 博客改造的个人（博客）站点。

### 仓库结构

```
qwz-site
├─site-backend  后端源码
│
├─site-frontend 前端源码
│  ├─site-frontend-admin 后台管理
│  ├─site-frontend-portal 前台门面
│ 
├─site-resource 项目资源
│  ├─img 图片
│  ├─jar 依赖包
```
